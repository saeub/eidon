## Experiment stages

| Experiment stage | Description |
| --- | --- |
| [`AudioCheck`](AudioCheck.md) | Shows a volume indicator to test the microphone. |
| [`Blank`](Blank.md) | Shows a blank screen. |
| [`CursorMultipleChoiceQuestion`](CursorMultipleChoiceQuestion.md) | Shows an image stimulus and allows selecting a response option by moving a cursor. |
| [`DriftCorrect`](DriftCorrect.md) | Starts the eye tracker's drift correction procedure (device-dependent). |
| [`EyeLinkDriftCorrect`](EyeLinkDriftCorrect.md) | Do not use this stage directly. Use DriftCorrect instead. |
| [`EyeLinkSetup`](EyeLinkSetup.md) | Do not use this stage directly. Use Setup instead. |
| [`FixationCross`](FixationCross.md) | Shows a fixation cross/trigger. |
| [`FreeTextQuestion`](FreeTextQuestion.md) | Shows an image stimulus and allows entering a free text response. |
| [`HostControlled`](HostControlled.md) | Disables interaction on the display PC and waits for commands from the host PC. |
| [`LabelAnnotation`](LabelAnnotation.md) | Shows an image stimulus and allows selecting a label. |
| [`MouseTrackerDriftCorrect`](MouseTrackerDriftCorrect.md) | Do not use this stage directly. Use DriftCorrect instead. |
| [`MouseTrackerSetup`](MouseTrackerSetup.md) | Do not use this stage directly. Use Setup instead. |
| [`MultipleChoiceQuestion`](MultipleChoiceQuestion.md) | Shows an image stimulus and allows selecting a response option by pressing a key. |
| [`QuestionTree`](QuestionTree.md) | Shows a series of question stages, where each stage is based on previous responses. |
| [`Setup`](Setup.md) | Starts the eye tracker's setup procedure (device-dependent). |
| [`StimulusMultiPage`](StimulusMultiPage.md) | Shows multiple image stimuli that can be navigated. |
| [`StimulusPage`](StimulusPage.md) | Shows a single image stimulus. |

> Can't find the experiment stage you need? [**Implement your own!**](custom.md)
