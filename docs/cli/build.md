## CLI command: `build`

Build an experiment from a configuration file and materials. Generates stimuli and session definitions that can be run with `eidon run`.

### Usage

```
[1;34musage: [0m[1;35mgenerate.py build[0m [[32m-h[0m] [[36m--area-images[0m] [32mpath[0m

Build an experiment from a configuration file and materials. Generates stimuli and session definitions that can be run with `eidon run`.

[1;34mpositional arguments:[0m
  [1;32mpath[0m           Path to the experiment directory (must contain config.yaml).

[1;34moptions:[0m
  [1;32m-h[0m, [1;36m--help[0m     show this help message and exit
  [1;36m--area-images[0m  Generate additional images with outlined areas of interest.
```
